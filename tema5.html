📦 Tema 05: Vetores em Portugol

Aprenda a armazenar múltiplos valores em uma única estrutura de dados

📖 O que são Vetores?

Imagine que você precisa armazenar as notas de 10 alunos. Criar 10 variáveis diferentes seria muito trabalhoso! Os vetores resolvem esse problema permitindo armazenar vários valores do mesmo tipo em uma única estrutura.

📌 Representação visual de um vetor:
Vetor Nomes:
[ Maria ] [ João ] [ Rosa ] [ Paula ] [ Fábio ]
   0       1       2       3       4

🔧 Declaração de Vetores

Sintaxe: tipo nome[tamanho]

Exemplos:
inteiro numeros[5]
real notas[10]
cadeia nomes[50]
logico aprovados[30]

📝 Inicialização e Acesso

Inicializando com valores:
inteiro vetor[5] = {10, 20, 30, 40, 50}
cadeia alunos[3] = {"Ana", "Bruno", "Carla"}

Acessando elementos (índice começa em 0):
escreva(vetor[0]) // mostra 10
vetor[2] = 100 // altera a posição 2 para 100

🔄 Vetores com Laços de Repetição

Lendo valores para o vetor:
para (inteiro i = 0; i < 5; i++) {
    escreva("Digite o valor ", i + 1, ": ")
    leia(vetor[i])
}

Exibindo os valores:
para (inteiro i = 0; i < 5; i++) {
    escreva("Posição ", i, ": ", vetor[i], "\n")
}
💡 Dica Importante:
• O primeiro índice do vetor é sempre 0 (zero)
• O último índice é tamanho - 1
• Valores padrão: inteiro/real = 0, caracter/cadeia = "", logico = falso
• Use laços "para" para percorrer vetores de forma eficiente
📌 Exemplo 1: Declaração e Acesso
✅ Declarando e acessando posições
programa
{
    funcao inicio()
    {
        inteiro numeros[5] = {15, 22, 8, 10, 11}
        cadeia nomes[3] = {"Ana", "Bruno", "Carla"}
        
        escreva("Primeiro número: ", numeros[0], "\n")
        escreva("Segundo número: ", numeros[1], "\n")
        escreva("Último número: ", numeros[4], "\n")
        
        escreva("\nPrimeiro nome: ", nomes[0], "\n")
        escreva("Segundo nome: ", nomes[1], "\n")
        escreva("Terceiro nome: ", nomes[2], "\n")
        
        numeros[2] = 100
        nomes[1] = "Beatriz"
        
        escreva("\n--- APÓS ALTERAÇÃO ---\n")
        escreva("Posição 2 agora é: ", numeros[2], "\n")
        escreva("Posição 1 agora é: ", nomes[1], "\n")
    }
}

📊 Exemplo 3: Média de Notas dos Alunos
✅ Vetores para nomes e notas
programa
{
    funcao inicio()
    {
        cadeia nomes[5]
        real av1[5]
        real av2[5]
        real media[5]
        cadeia pausa
        
        para (inteiro i = 0; i < 5; i++) {
            escreva("\n--- ALUNO ", i + 1, " ---\n")
            escreva("Nome: ")
            leia(nomes[i])
            escreva("Nota AV1: ")
            leia(av1[i])
            escreva("Nota AV2: ")
            leia(av2[i])
            media[i] = (av1[i] + av2[i]) / 2
        }
        
        escreva("\n\n=== MÉDIA DOS ALUNOS ===\n")
        para (inteiro i = 0; i < 5; i++) {
            escreva("Nome: ", nomes[i], 
                    " Nota AV1: ", av1[i],
                    " Nota AV2: ", av2[i],
                    " Média: ", media[i], "\n")
        }
        
        escreva("\nPressione ENTER para continuar...")
        leia(pausa)
    }
}

🔢 Exemplo 4: Filtrar Números Pares
✅ Processamento com vetores
programa
{
    funcao inicio()
    {
        inteiro numeros[10]
        inteiro pares[10]
        inteiro qtdPares = 0
        cadeia pausa
        
        para (inteiro i = 0; i < 10; i++) {
            escreva("Digite o ", i + 1, "º número: ")
            leia(numeros[i])
            
            se (numeros[i] % 2 == 0) {
                pares[qtdPares] = numeros[i]
                qtdPares = qtdPares + 1
            }
        }
        
        escreva("\n=== NÚMEROS PARES ===\n")
        se (qtdPares == 0) {
            escreva("Nenhum número par foi digitado.\n")
        }
        senao {
            para (inteiro i = 0; i < qtdPares; i++) {
                escreva(pares[i], " ")
            }
            escreva("\n")
        }
        
        escreva("\nTotal de números pares: ", qtdPares, "\n")
        escreva("\nPressione ENTER para continuar...")
        leia(pausa)
    }
}

🏆 Exercícios Propostos

# Exercício Descrição
1 Vetor de 5 números Desenvolva um programa que armazene 5 números inteiros digitados pelo usuário e depois os exiba em tela.
2 Números Aleatórios Crie um programa que gere 5 números aleatórios (0 a 100) e os armazene em um vetor, depois exiba todos.
3 Média dos Alunos Faça um programa que leia nomes, notas AV1 e AV2 de 5 alunos, calcule a média e exiba os resultados.
4 Números Pares Leia 10 números inteiros e exiba apenas os números pares armazenados no vetor.
5 Maior e Menor Crie um programa que leia 8 números e exiba o maior e o menor valor encontrado.

💡 Dicas para os exercícios

📌 Dicas Gerais

  • Percorrer vetores: use para (inteiro i = 0; i < tamanho; i++)
  • Índices: o primeiro índice do vetor é sempre 0, o último é tamanho - 1
  • Pausa: use cadeia pausa e leia(pausa) para aguardar o ENTER

🔢 Exercício 1 - Vetor de 5 números

  • Declare o vetor: inteiro numeros[5]
  • Use um laço para para ler cada posição
  • Use outro laço para para exibir os valores
Estrutura básica:
para (inteiro i = 0; i < 5; i++) {
    escreva("Digite o ", i + 1, "º número: ")
    leia(numeros[i])
}

🎲 Exercício 2 - Números Aleatórios

  • Importante: inclua a biblioteca Util no início do programa
programa
{
    inclua biblioteca Util
    funcao inicio()
    {
        // seu código aqui
    }
}
  • Gere números aleatórios com: Util.sorteia(0, 100)
  • A função sorteia(min, max) retorna um número inteiro entre min e max (inclusive)
Exemplo:
numeros[i] = Util.sorteia(0, 100)

📊 Exercício 3 - Média dos Alunos

  • Crie 4 vetores: cadeia nomes[5], real av1[5], real av2[5], real media[5]
  • Calcule a média: media[i] = (av1[i] + av2[i]) / 2
  • Exiba os resultados formatados

🔢 Exercício 4 - Números Pares

  • Verifique se um número é par: numeros[i] % 2 == 0
  • Use um contador para controlar a posição no vetor de pares
  • Exiba apenas os números que atendem à condição

📈 Exercício 5 - Maior e Menor

  • Inicialize maior e menor com o primeiro elemento: maior = numeros[0]
  • Percorra o vetor a partir do índice 1
  • Compare cada elemento e atualize maior e menor quando necessário

📋 Estrutura Básica para os Exercícios

programa
{
    funcao inicio()
    {
        // Declaração das variáveis e vetores
        cadeia pausa
        
        // Entrada de dados
        para (inteiro i = 0; i < tamanho; i++) {
            escreva("Digite o valor: ")
            leia(vetor[i])
        }
        
        // Processamento (quando necessário)
        
        // Saída de dados
        escreva("\n=== RESULTADO ===\n")
        para (inteiro i = 0; i < tamanho; i++) {
            escreva(vetor[i], " ")
        }
        escreva("\n")
        
        // Pausa
        escreva("\nPressione ENTER para finalizar...")
        leia(pausa)
    }
}
